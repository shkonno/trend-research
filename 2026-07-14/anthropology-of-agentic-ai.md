### 5. Managing Procedural Memory in LLM Agents: Control, Adaptation, and Evaluation
- 出典: arXiv
- 日付: 2026-06-22（直近14日より少し古いが、関連性が高いため採用）
- リンク: https://arxiv.org/abs/2606.23127
- 要約: LLMエージェントの「手続き記憶」が、繰り返し発生する職場タスクで再利用可能なスキルを生むかを評価する研究。382件の現実的な企業タスク、6つの専門職ロール、22の手続きスキルを含むAFTERベンチマークを提示し、スキルの役割横断・タスク横断・モデル横断の移転を調べている。
- なぜ面白いか:
  - 技術: エージェントの改善を単発の推論性能ではなく、反復業務から抽出される手続きスキルとその移転可能性として評価している。
  - 人文: 「手続き記憶」は、職場の慣習や新人教育で受け継がれる“やり方”に近い。エージェントが記憶するスキルが部署や役割を越えて通用するかという問いは、組織文化の翻訳可能性を測る人類学的な問いにもなっている。

## arXiv / 学術
- Managing Procedural Memory in LLM Agents: Control, Adaptation, and Evaluation — arXiv:2606.23127。職場タスクにおける手続き記憶とスキル移転を扱うため、本日のトップ5に採用。
- AGORA: An Archive-Grounded Benchmark for Agentic Workplace Document Reasoning — arXiv:2606.24526。乱雑な職場文書アーカイブから証拠を探し、用語や単位の不一致を調整するエージェント評価で、組織の記録文化との接点がある。
- EnterpriseClawBench: Benchmarking Agents from Real Workplace Sessions — arXiv:2606.23654。実際の職場セッションに基づく企業エージェント・ベンチマークで、現場データに近い評価軸として重要。
- PolyWorkBench: Benchmarking Multilingual Long-Horizon LLM Agents — arXiv:2607.06008。多言語・長期タスクの評価で、英語中心ではないローカル文化のエージェント運用を考える補助線になる。
