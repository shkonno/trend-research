### 5. CodeRescue: Budget-Calibrated Recovery Routing for Coding Agents
- 出典: arXiv
- 日付: 2026-07-21
- リンク: https://arxiv.org/abs/2607.19338v1
- 要約: Coding agentが失敗した後、安いモデルで追加リカバリを続けるべきか、高価で強いモデルへエスカレーションすべきかを、実行フィードバックと予算制約に基づいてルーティングする研究です。Conformal Risk Controlを使い、予算に応じたコスト制御を行いながら、常に強いモデルへ投げるより低コストで高い解決率を狙います。
- なぜ面白いか:
  - 技術: Claude Codeのような実行環境付きエージェントでは、失敗は単なる不正解ではなく次の行動選択の材料になるため、モデル選択・再試行・エスカレーションの制御が重要になります。
  - 人文: 「いつ諦め、いつ助けを呼ぶか」は人間の仕事術にも近い判断です。AIエージェントの経済性を考えることは、労働時間、計算資源、品質保証をどう配分するかという組織の価値判断にもつながります。
